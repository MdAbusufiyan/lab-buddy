# LAB Buddy

LAB Buddy is a lightweight desktop application designed for chemical lookup and logging in laboratory environments.

The application retrieves chemical information from PubChem (NIH) and supports offline usage through a local cache, making it suitable for laboratories with limited or unreliable internet access.

## Features
- Online chemical search via PubChem
- Offline access using local cache
- Display of chemical identifiers and properties
- Structure image display
- GHS hazard pictograms and hazard statements (limited)
- Excel (.xlsx) logging

## Platform
- Windows (precompiled executable provided)
- Python source code available

## Data Source
Chemical data and structure images are sourced from PubChem (NIH).
